# 🛒 Kiosquito

Kiosquito es un proyecto personal desarrollado para la gestión de kioscos y pequeños comercios. Permite administrar el inventario, registrar ventas, consultar el historial de operaciones y acceder al sistema desde distintos dispositivos. Además, incorpora la opcion de una suscripción mensual, con funcionalidades avanzadas como reportes, estadísticas y gráficos de ventas
